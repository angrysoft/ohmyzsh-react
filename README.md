# react plugin

React.js helpers

Created by [AngrySoft](https://github.com/angrysoft)

## Usage

Add the plugin to the plugins array in your zshrc file and restart zsh:

```zsh
plugins=(... react)
```

`react` for react helpers
`create_react_app`

## Aliases

| Alias   | Command                            |
| ------- | ---------------------------------- |
| cra     | `create_react_app`                 |
